# Employee Management API

A simple REST API built using FastAPI and SQLAlchemy for managing employees.

## Features
- Create employee
- Read all employees
- Get employee by ID
- Update employee
- Delete employee

## Tech Stack
- FastAPI
- SQLAlchemy
- SQLite / PostgreSQL
- Python

## How to run locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
